Callback / Функции обратного вызова

Параметр и аргумент

const double = (param) => param + param;
double(argument);

Функция принимает данные, преобразовывает данные и возвращает данные.

В JavaScript вместо данных функции можно передать другую функцию как аргумент, ведь та в свою очередь все равно вместо себя подставит данные. 

const log = (str) => console.log(str);  
log(double("a"));  // aa

Функция как параметр и аргумент

Также JavaScript вместо данных можно указать на применение какой-то функции как параметр. А при вызове подставить конкретную функцию как аргумент. В итоге все равно будут передаваться вычисленные данные. 


const  add = param => param + param;
const  mult = param => param * param;
const multFoo = (num, foo) => num * foo(num);
console.log(multFoo(3, add));  // 18
console.log(multFoo(3, mult));  // 27

Функция, как значение массива

Этот пример показывает, что функция может быть значением массива и использоваться в вычислениях.

Позже мы увидим, что массив это разновидность объекта, а функция внутри объекта называется метод. 


const array = [
    a => a + a,
    a => a * a,
    a => a / a,
    a => a**a
]      
const ariphmetic = (arr, val) => arr.map(elem => elem(val)); 
console.log(ariphmetic(array, 3)); // [6, 9, 1, 27]

Практическое использование
