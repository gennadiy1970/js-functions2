Arrow function

Синтаксис

const sum = function (a, b) {
            return a + b;            
        }

Все, отмеченное фоном у обычной функции      заменяется на => у стрелочной функции:

  • function заменятся на =>
  • без  {  и  }  все что после => будет return
Вариант без скобок и return
const sum = (a, b)  => a + b;
console.log(sum(2, 3))   // 5
Вариант cо скобками и return

При наличии скобок обязатльно нужно ставить return, иначе вернется undefined

const sum = (a, b)  => { a + b; }
console.log(sum(2, 3))   // undefined
const add = (a, b)  => { return a + b; }
console.log(add(2, 3))   // 5
Вариант без аргументов

Обязательно требуютя пустые скобки

const ok = ()  => "It's ok!";
console.log(ok())   // "It's ok!"
Вариант с одним аргументом

Скобки не обязательны

const add2 = a => a + 2;
console.log(add2(3))   // 5

Три "нет" Arrow function

  1. Отсутствует arguments
  2. Не имеет собственного this // this по месту создания
  3. Не рабтает с new и как следствие не рабоатет и с super
